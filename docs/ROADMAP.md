## Phase 6 — 콘텐츠 확장 + 셰프 캐릭터

> 목표: 재료 풀 확장, 레시피 42종 달성, 셰프 시스템으로 전략 다양성 확보

### 6-1. 신규 재료/적 + 캠페인 확장

- [ ] 재료 3종 추가: 생선(🐟), 버섯(🍄), 치즈(🧀)
- [ ] 적 3종 추가: fish_knight, mushroom_scout, cheese_rat
- [ ] 스테이지 2-1 ~ 2-3 추가 (2장: 동양 요리 식당 테마)
- [ ] 2장 보스 적 1종

### 6-2. 레시피 확장 (+18종)

- [ ] 서빙 +15종 (★ 버섯 수프, 구운 생선, 치즈 토스트 / ★★ 피시앤칩스, 생선초밥, 버섯 리조또, 치즈 오믈렛 / ★★★ 해산물 그라탕, 버섯 스테이크, 치즈 퐁뒤 그라탱 / ★★★★ 풀코스 해산물 등)
- [ ] 버프 +3종 (치즈 쉴드: 피해 -20%, 버섯 집중: 사거리 +30%, 생선 오일: 속도 +35%)
- [ ] 스테이지 게이트 연동 (2장 클리어 → ★★★★ 레시피 해금)

### 6-3. 셰프 캐릭터 시스템

- [ ] 3종 셰프 캐릭터 (선택 UI)
  - 꼬마 셰프: 재료 수거 범위 +30%, 스킬 "긴급 배달"
  - 불꽃 요리사: 화염 타워 +20%, 스킬 "지옥 불꽃"
  - 얼음 요리장: CC +25%, 스킬 "블리자드 코스"
- [ ] 셰프 선택 씬 (스테이지 시작 전)
- [ ] 패시브 효과 게임 내 반영
- [ ] 스킬 버튼 UI + 쿨다운

### 6-4. 오더 시스템

- [ ] 특정 웨이브에 조건부 미션 ("해산물 요리 2개 완성 → 보너스 골드")
