#### Phase 78 — 이진 실패 → 부분 성공 구조 전환 (full, none)

**배경**: TD 전투에서 HP가 0이 되면 재료가 0개가 되어 영업 씬 전체가 건너뛰어진다. 한 번 실패하면 보상이 전혀 없어 유저 이탈이 급격히 발생한다.

**목표**: HP 0 도달 후에도 "부분 성공" 상태로 영업 씬 진입 허용. 수집한 재료 비율에 비례해 서빙 가능 레시피 풀을 제한.

**핵심 메커닉**:
```
HP > 0 (기존 클리어): 전체 수집 재료 → 정상 영업
HP = 0, 재료 > 0 (부분 실패): 수집 재료의 50%만 영업에 사용. 별점 최대 ★★☆
HP = 0, 재료 = 0 (완전 실패): 기존 "장보기 실패" 화면 유지
```

**구현 범위**:
- `GatheringScene.js`: HP=0 종료 시 `partialFail: true` 플래그 ServiceScene에 전달
- `ServiceScene.js`: `partialFail` 시 인벤토리 아이템 수를 ceil(50%) 컷
- `ResultScene.js`: `partialFail` 상태에서 별점 최대 2개 캡 적용, 결과 메시지 분기
- `marketResult` 전달 데이터에 `partialFail: boolean` 추가

**의존성**: 없음.
