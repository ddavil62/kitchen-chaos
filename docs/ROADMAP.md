## Phase 4 — 밸런스 + 스테이지 시스템

> 목표: 1스테이지짜리 프로토를 "여러 스테이지를 진행하는 게임"으로 전환

### 4-1. 밸런싱 및 튜토리얼 웨이브

- [ ] 웨이브 시작 전 준비 시간 확보 (5~8초, 현재 2.6초)
- [ ] 웨이브 1~3 난이도 완화 (적 수 감소, 간격 확대)
- [ ] "다음 웨이브 시작" 수동 버튼 (자동 진행 대신 플레이어 선택)
- [ ] 간단 튜토리얼 오버레이 (첫 플레이 시 타워 배치/서빙 가이드)
- [ ] 밸런스 수치 조정 (골드, 적 HP/속도, 타워 DPS)

### 4-2. 스테이지 선택 시스템

- [ ] 스테이지 선택 씬 (`StageSelectScene`)
- [ ] 스테이지 데이터 구조 (`stageData.js` — 맵, 경로, 웨이브, 손님 구성)
- [ ] 3개 스테이지 (1-1 ~ 1-3, 파스타 레스토랑 테마)
- [ ] 각 스테이지별 고유 경로/맵 레이아웃
- [ ] 별점 시스템 (1~3★, 남은 생명 기반)
- [ ] 스테이지 잠금/해금 (이전 스테이지 클리어 시 해금)

### 4-3. 세이브 시스템

- [ ] localStorage 세이브 (스테이지 진행도, 별점, 누적 골드)
- [ ] 세이브 버전 관리 (마이그레이션 체인)

### 4-4. 미완성 적/타워 추가

- [ ] 밀가루 유령 (6번째 적 — 투명 이동, 특수 타워만 공격 가능)
- [ ] 냉동고 타워 (5번째 — CC 빙결)
- [ ] 수프 솥 타워 (6번째 — 범위 치유/재생)

**구현 단위**: 4-1 → 4-2 → 4-3 → 4-4

## Phase 5 — 메타 진행 시스템

> 목표: "클리어하면 끝"이 아닌 반복 플레이 동기 부여

### 5-1. 주방 코인 + 영구 업그레이드

- [ ] 주방 코인 (별점 → 코인 변환)
- [ ] 영구 업그레이드 5종
  - 냉장고 확장 (재료 최대 보유량 +2)
  - 예리한 칼 (모든 타워 공격속도 +5%)
  - 빠른 배달 (재료 수거 속도 +20%)
  - 조리 특훈 (조리 시간 -10%)
  - 레시피북 (레시피 힌트 1개 공개)
- [ ] 업그레이드 상점 씬 (`UpgradeShopScene`)
- [ ] 업그레이드 효과가 게임 내 실제 반영

### 5-2. 캠페인 확장 (6 스테이지)

- [ ] 스테이지 1-4 ~ 1-6 추가 (파스타 레스토랑 완성)
- [ ] 보스 웨이브 (1-6 마지막 — 특수 패턴 보스 적)
- [ ] 테마별 배경색/타일 차이

### 5-3. 평판 수치 UI

- [ ] 스테이지 결과 화면 개선 (별점 + 통계)
- [ ] 누적 평판 표시 (메인 메뉴)

**구현 단위**: 5-1 → 5-2 → 5-3

## Phase 6 — 셰프 캐릭터 + 레시피 발견

> 목표: 전략적 다양성과 재발견의 재미

### 6-1. 셰프 캐릭터 시스템

- [ ] 3종 셰프 캐릭터 (선택 UI)
  - 꼬마 셰프: 재료 수거 범위 +30%, 스킬 "긴급 배달"
  - 불꽃 요리사: 화염 타워 +20%, 스킬 "지옥 불꽃"

- [ ] 셰프 선택 씬 (스테이지 시작 전)
- [ ] 패시브 효과 게임 내 반영
- [ ] 스킬 버튼 UI + 쿨다운

### 6-2. 레시피 발견 시스템

- [ ] 미공개 레시피 (??? 표시) → 조합 시 최초 발견
- [ ] 레시피북 UI (발견된/미발견 목록)
- [ ] 발견 이벤트 연출 (팝업 + 보상)
