## 1. 풀 확대 원칙

### 1-1. 목표 인원: 총 28명

기존 8명(Phase 48-2)에 신규 20명을 추가하여 총 28명으로 확정한다.

**28명 근거:**
- 도감(CollectionScene) 운영 최소 규모: 화면을 스크롤 없이 그리드 4열 기준 약 24~32칸이 적정 분량.
- 희귀도별 분포(아래 참조)가 고르게 채워지는 숫자이면서, 플레이어가 전 등급을 모두 수집하는 목표 달성까지 수 주~수 개월의 장기 수집 동기를 제공한다.
- 세계관의 요리 아크(7개 테마) × 4등급으로 구성하면 28개가 자연스러운 배분 단위.

### 1-2. 등급별 배분 (기존 + 신규 합산)

| 등급 | 기존 인원 (Phase 48-2) | 신규 추가 | 합계 | 비율 |
|------|----------------------|---------|------|------|
| 초급 (1등급) | 2명 | 5명 | 7명 | 25% |
| 중급 (2등급) | 3명 | 7명 | 10명 | 36% |
| 고급 (3등급) | 2명 | 5명 | 7명 | 25% |
| 전설 (4등급) | 1명 | 3명 | 4명 | 14% |
| **합계** | **8명** | **20명** | **28명** | 100% |

**설계 근거:**
- 초급·중급이 전체 61%를 차지해 진입 초기 플레이어가 쉽게 수집할 수 있다.
- 전설은 4명으로 희소성을 유지한다. 수집 완료 목표 시 전설 4명 확보가 마지막 관문이 된다.
- 고급 7명은 중반 이후 콘텐츠 동기 부여 역할.

### 1-3. 세계관 지역별 배분 (요리 아크 연계)

**그리드 사양:**

## 4. 획득 다양화 — 타겟 획득 경로 보완

### 4-1. 현행 4개 경로의 한계

Phase 48-1에서 확정한 4개 경로(미력 나그네 서빙/보스 클리어/스토리 이정표/엔드리스 이정표)는 모두 "랜덤 고용권 획득" 구조다. 풀이 28명으로 확대되면 원하는 특정 미력사를 타겟팅하기 어려워진다.

### 4-2. 타겟 고용 메커닉 — 아크 친화 서빙

**규칙:**
> 특정 요리 아크의 레시피를 해당 미력 나그네에게 서빙하면, 그 아크 소속 미력사 중 미해금 미력사의 고용권이 활성화된다.

| 조건 | 효과 |
|------|------|
| 미력 나그네에게 한식 레시피 서빙 | 한식 아크 미력사 중 1명 타겟 고용 가능 상태 전환 (1회용) |
| 미력 나그네에게 일식 레시피 서빙 | 일식 아크 미력사 타겟 고용 가능 전환 |
| (이하 아크별 동일 패턴) | — |

**타겟 고용 가능 상태:**
- 해당 아크에 미해금 미력사가 남아있을 때만 발동.
- "고용 가능" 상태가 되면 WanderingChefModal에 해당 미력사가 흐릿한 실루엣에서 컬러로 전환되며 [타겟 고용] 버튼 활성화.
- [타겟 고용] 버튼은 3일(실시간 기준) 동안 유효. 이후 만료 시 다시 조건 충족 필요.

**구현 고려사항:**
- 세이브에 `wanderingChefs.targetHireSlots: { [arcId]: { chefId, expireTime } }` 필드 추가.
- 만료 타이머는 세션 시작 시 체크.

### 4-3. 미력사 교환 (교환 패드) — 도입 여부

| 방안 | 의견 |
|------|------|
| 미력사 교환 | **채택하지 않음.** 미력의 정수가 희귀 화폐인 만큼 "잘못 샀다"는 실수를 허용하면 희귀성이 훼손된다. 대신 재고용 할인(50%)과 타겟 고용 메커닉으로 보완. |
| 미력사 합성 | **채택하지 않음.** 합성 시스템은 복잡도를 과도하게 높인다. 단순 수집 → 강화 구조를 유지한다. |
