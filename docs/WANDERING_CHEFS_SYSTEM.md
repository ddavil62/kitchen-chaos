## 1. 세계관 및 개념 정의

### 1-1. 유랑 미력사란

**유랑 미력사(流浪 美力師)**는 식란이 발생하는 각지를 떠돌며 자신의 요리 미력을 연마하는 독립 요리사들이다. 이들은 특정 가문이나 문파에 소속되지 않고, 오로지 자신의 경험과 직관으로 미력을 다루는 길을 택한 이들이다.

미력사의 세계에서는 이들을 "뿌리 없는 칼"이라 부르기도 한다. 소속과 전통이 없는 대신, 어느 환경에서도 요리할 수 있는 적응력과 독창성이 그들의 강점이다.

### 1-2. 어디서 오는가

유랑 미력사들은 미력의 정수를 따라 이동한다. 정수는 격렬한 식란이 일어난 자리, 혹은 오랜 세월 수많은 손님을 만족시킨 식당에 자연스럽게 축적된다. 미미의 식당은 할머니 때부터 이어온 정화의 역사가 있어, 정수가 서서히 모이는 장소가 되어가고 있다.

> "정수 냄새가 나더군요. 이 식당, 예사롭지 않아요."
> — 유랑 미력사 첫 방문 시 공통 대사

### 1-3. 어떻게 합류하는가

미력 나그네(Phase 48-1의 특수 손님 NPC)와 달리, 유랑 미력사는 고용 계약을 통해 식당에 합류한다. 미미가 보유한 미력의 정수를 지불하면 일정 기간 또는 영구적으로 함께 일한다. 이들은 전투(식란 정화)가 아닌 **영업 특화** 역할을 담당하며, 스토리 셰프 5종(꼬마/불꽃/얼음/유키/라오)의 전투 역할과 명확히 분리된다.

### 1-4. 스토리 셰프와의 역할 분리

| 구분 | 스토리 셰프 (5종) | 유랑 미력사 |
|------|-----------------|------------|
| 역할 | 식란 정화 (전투/채집 특화) | 영업 특화 |
| 획득 | 스토리 진행 (무료) | 미력의 정수 소비 (고용) |
| 스킬 영향 범위 | GatheringScene 전투 메커닉 | ServiceScene 영업 메커닉 |
| 동시 활용 | 스테이지당 1명 선택 | 고용 상한 내 다중 동시 활용 |
| 해고/교체 | 불가 (영구 파티 멤버) | 가능 (단, 정수 환급 없음) |

**픽셀아트 컨셉**: 보라색 단발에 하늘거리는 흰 조리복. 허리에 은색 띠. 눈에 옅은 보랏빛 오라가 감도는 이펙트. 자신감 넘치는 눈빛과 가볍게 미소짓는 입꼬리. 32x48px.

## 4. 고용 UI 흐름

### 4-1. 진입점

유랑 미력사 고용 화면은 **상점(ShopScene)** 5번 탭 "직원" 탭에서 진입한다. 현재 직원 탭에는 서빙 도우미(waiter)와 세척 도우미(dishwasher)가 있으며, 그 아래에 "유랑 미력사 고용" 섹션을 추가한다.

| 탭 | 기존 항목 | 추가 항목 |
|----|---------|---------|
| 직원 탭 | 서빙 도우미, 세척 도우미 | 유랑 미력사 고용 버튼 (별도 서브 화면으로 이동) |

미력사 고용 버튼을 누르면 ShopScene 위에 모달 형태의 `WanderingChefModal`이 열린다.

### 4-2. 화면 구성 (WanderingChefModal)

```
┌─────────────────────────────────────┐
│  유랑 미력사    [보유 정수: 💠 N]   │  ← 헤더 + 현재 정수 표시
├─────────────────────────────────────┤
│  [등급 필터: 전체 / 초급 / 중급 / 고급 / 전설]  │
├─────────────────────────────────────┤
│  미력사 카드 목록 (스크롤)           │
│  ┌────────┐  ┌────────┐            │
│  │ 캐릭터 │  │ 이름   │            │
│  │ 스프라│  │ 칭호   │            │
│  │ 이트  │  │ 등급   │            │
│  │       │  │ 스킬명 │            │
│  │       │  │ [상세] │            │
│  └────────┘  └────────┘            │
│  [고용: 💠 N] / [고용 중] / [해금 안 됨] │
├─────────────────────────────────────┤
│  현재 고용 중: N/M명               │  ← 동시 고용 상한 표시
└─────────────────────────────────────┘
```

- **카드 상태**: 미해금(잠금 아이콘 + 흐릿 표시) / 해금 가능(고용 버튼 활성) / 고용 중(강화 버튼 + 해고 버튼)
- **상세 버튼**: 스킬 효과 전문 설명 팝업 오픈
- **고용 버튼**: 정수 부족 시 비활성 + "💠 N 부족" 텍스트

### 4-5. 해고 규칙

- 언제든지 해고 가능.
- 해고 시 미력의 정수 환급 **없음** (Phase 48-1 결정 준수).
- 해고해도 도감에서는 해금 상태 유지 (재고용 가능).
- 재고용 시 고용 비용이 아닌 **재고용 비용** 적용 (등급별 50% 할인).
- 강화 단계는 해고 후에도 보존됨 (재고용 시 이전 강화 단계 복원).

## 5. 강화 시스템

### 5-1. 강화 개요

| 항목 | 내용 |
|------|------|
| 강화 재화 | 미력의 정수 |
| 강화 단계 | 1단계 → 2단계 → 3단계 (최대 3단계) |
| 강화 조건 | 해당 미력사가 현재 고용 중이어야 함 |
| 강화 취소 | 불가. 강화는 영구 적용. |

### 5-2. 강화 비용 (Phase 48-1 확정값)

| 등급 | 1→2단계 | 2→3단계 | 총 투자량 (고용+전강화) |
|------|--------|--------|----------------------|
| 초급 | 3 정수 | 6 정수 | 4 + 3 + 6 = 13 정수 |
| 중급 | 10 정수 | 20 정수 | 12 + 10 + 20 = 42 정수 |
| 고급 | 30 정수 | 60 정수 | 35 + 30 + 60 = 125 정수 |
| 전설 | 70 정수 | 140 정수 | 80 + 70 + 140 = 290 정수 |

### 5-3. 강화 UI

고용 중인 미력사 카드 내에 강화 버튼 표시:

```
[고용 중] 레벨: ★★☆ (2/3단계)
[강화: 💠 60] ← 비용 실시간 표시. 정수 부족 시 비활성.
```

```javascript
// ── Phase 48-2 추가 필드 ──

// 미력의 정수 화폐 (Phase 48-1에서 설계, Phase 48-2 구현)

// 유랑 미력사 시스템
data.wanderingChefs = {
  hired: [],           // 현재 고용 중인 미력사 ID 목록 (예: ['wanderer_haruka'])
```

### 7-2. v17 → v18 마이그레이션 로직

```javascript
// v17 → v18: 미력의 정수 화폐 + 유랑 미력사 시스템 추가 (Phase 48-2)
if (data.version < 18) {
  // 미력의 정수 화폐 필드
  if (data.mireukEssence === undefined)      data.mireukEssence = 0;
  if (data.mireukEssenceTotal === undefined) data.mireukEssenceTotal = 0;
  if (data.mireukTravelerCount === undefined) data.mireukTravelerCount = 0;
  if (data.mireukBossRewards === undefined)  data.mireukBossRewards = {};

  // 유랑 미력사 시스템 필드
  if (!data.wanderingChefs) {
    data.wanderingChefs = {
      hired: [],
      unlocked: [],
      enhancements: {},
    };
  }

  data.version = 18;
}
```

## 8. ServiceScene 연동 설계

### 8-1. 스킬 적용 시점

유랑 미력사 스킬은 ServiceScene 초기화(`create()`) 시점에 `SaveManager.getWanderingChefs()`를 호출하여 고용 중인 미력사 목록을 로드하고, 각 스킬을 해당 씬의 버프 변수에 반영한다. 스킬은 씬 전체 기간 동안 지속된다.

### 8-2. 스킬 적용 변수 매핑

| 미력사 스킬 | 적용 대상 변수/상수 | 적용 방식 |
|------------|------------------|---------|
| 친절 미소 (하루카) | `CUSTOMER_PATIENCE_MULT` | 전 손님 유형 인내심 배율 +N% |
| 여유 분위기 (보태) | 조리 완료 타이머 | cookTime × (1 - 감소율) |
| 이국의 풍미 (레이라) | `SPECIAL_CUSTOMER_RATES.gourmet` | 미식가 확률 상향 + 보상 배율 상향 |
| 창식 서빙 (무오) | 서빙 처리 딜레이 | 서빙 상태 전환 딜레이 × (1 - 감소율) |
| 단골 확보 (시엔) | 세션 초반 30~60초 보상 배율 | timer 기반 조건부 reward_mult 상향 |
| 결정 환불 (아이다) | 조리 실패 콜백 | 실패 시 재료 N% 인벤토리 복원 |
| 귀족의 향연 (로살리오) | `SPECIAL_CUSTOMER_RATES.vip` | VIP 확률 × N배, VIP 보상 배율 상향 |
| 연쇄 폭풍 서빙 (요코) | 연속 서빙 카운터 + 퇴장 방지 플래그 | N연속 달성 시 다음 손님 patience = Infinity (1회) |

### 8-3. 다중 미력사 스킬 중복 적용

동일 효과 유형의 스킬이 여러 미력사에게서 중복 발동될 경우 (예: 인내심 보너스 미력사 2명 동시 고용) **덧셈 합산**을 원칙으로 한다.

## 10. 밸런스 검토 포인트

### 10-1. 정수 소비 시뮬레이션

| 플레이어 목표 | 필요 정수 | 예상 확보 시간 |
|------------|---------|-------------|
| 초급 1명 고용 | 4 | 40분~1시간 |
| 초급 1명 만강화 | 4 + 9 = 13 | 약 2~3시간 |
| 중급 1명 고용 | 12 | 약 3시간 |
| 전설 1명 고용 | 80 | 24장 클리어 근접 시점 |
| 전설 1명 만강화 | 80 + 210 = 290 | 엔드리스 포함 장기 플레이 |

### 10-2. 고용 상한 연동 밸런스

동시 고용 가능 인원이 증가하는 시점(7→13→19장)에 맞춰 중급·고급 미력사의 해금 조건도 해당 구간에 분포하도록 설계했다. 플레이어가 "더 고용하고 싶다"는 욕구와 "정수가 부족하다"는 희소성이 병존하도록 유지한다.

### 10-3. 구현 후 조정 대상

- 퇴장 방지(요코 3단계) 효과가 영업 긴장감을 과도하게 해소할 경우: 발동 조건 강화 (3연속 → 5연속) 또는 발동 횟수 세션당 1→2회로 제한.
- 레이라·로살리오 중복 고용 시 VIP/미식가 확률이 지나치게 높아지는 경우: 개별 확률에 캡(최대 50%) 적용.
